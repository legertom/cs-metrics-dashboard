# CS Metrics Dashboard

Live Talkdesk metrics — Call Acceptance Rate, Schedule Adherence, and QA Score — for every agent and the team, with a 26-week heatmap, monthly charts, and a filterable agent list.

## Using this project with Claude Code

Clone the repo, open the folder in Claude Code, and ask it to help you. Some things you can say:

- *"Set up the project and run it with sample data"*
- *"I have my Talkdesk credentials — help me connect to live data"*
- *"Open the app in my browser"*
- *"Show me which agents are below the 85% target this month"*
- *"Add [agent name] to the roster"*

Claude Code will read `CLAUDE.md` in this folder to understand how everything works.

## Tech stack

Next.js 15 · Tailwind CSS · Recharts · TypeScript
